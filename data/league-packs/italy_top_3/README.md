# Italy Top 3 League Pack

Sidecar reference-data for the experimental `italy_top_3` league pack.

Status:

- not active in production UI by default
- covers Serie A, Serie B and provided Serie C group fixtures
- built as a premium country pack behind entitlement access

The league structure follows the professional pyramid of Serie A, Serie B and Serie C described in the Italian football league system reference:

- https://en.wikipedia.org/wiki/Italian_football_league_system
